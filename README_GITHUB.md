# Forex Monitor - تطبيق مراقبة الفوركس

تطبيق أندرويد لمراقبة 10 أزواج فوركس رئيسية باستخدام مؤشرات StochRSI و MACD.

## الميزات

- ✅ مراقبة 10 أزواج فوركس
- ✅ مؤشرات StochRSI و MACD
- ✅ إشعارات محلية فورية
- ✅ فحص تلقائي يومي عند 00:05 UTC
- ✅ واجهة عربية كاملة
- ✅ قاعدة بيانات محلية

## بناء APK تلقائياً

يتم بناء APK تلقائياً عند كل push إلى المستودع باستخدام GitHub Actions.

### تحميل APK

1. اذهب إلى تبويب **Actions** في المستودع
2. اختر آخر workflow ناجح
3. حمّل ملف APK من **Artifacts**

أو:

1. اذهب إلى تبويب **Releases**
2. حمّل آخر إصدار

## التطوير المحلي

```bash
pip install -r requirements.txt
python main.py
```
