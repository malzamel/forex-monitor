# تطبيق مراقبة الفوركس - Forex Monitor

تطبيق أندرويد لمراقبة 10 أزواج عملات رئيسية باستخدام مؤشرات StochRSI و MACD.

## المميزات

- ✅ مراقبة 10 أزواج فوركس رئيسية
- ✅ مؤشرين تقنيين: StochRSI (دقة عالية) و MACD (تأكيد)
- ✅ إشعارات محلية فورية عند ظهور إشارات جديدة
- ✅ واجهة عربية كاملة مع دعم RTL
- ✅ تصميم نظيف مستوحى من الموقع الرسمي
- ✅ فحص يومي تلقائي عند 00:05 UTC
- ✅ قاعدة بيانات محلية SQLite
- ✅ عرض نسب الدقة وعوامل الربح التاريخية

## التثبيت والتشغيل

### 1. تثبيت المتطلبات

```bash
pip install kivy kivymd requests plyer schedule
```

### 2. تشغيل التطبيق (للتطوير)

```bash
python3 main.py
```

### 3. بناء APK للأندرويد

```bash
# تثبيت buildozer
pip install buildozer

# بناء APK
buildozer android debug
```

سيتم إنشاء ملف APK في المجلد: `bin/`

## الاستخدام

### الشاشة الرئيسية

- عرض بطاقات الإحصائيات (عدد الأزواج، إشارات اليوم، الأنظمة النشطة)
- قائمة الإشارات الحديثة مع التفاصيل
- زر التحديث اليدوي
- الوصول للإعدادات

### الإشعارات

- إشعار فوري عند ظهور إشارة جديدة
- منع التكرار (لا ترسل نفس نوع الإشارة مرتين)
- عرض السعر والدقة وعامل الربح

### الفحص التلقائي

- يتم الفحص اليومياً عند 00:05 UTC (3:05 صباحاً بتوقيت السعودية)
- فحص جميع الأزواج العشرة
- إرسال إشعارات للإشارات الجديدة فقط

## مصدر البيانات

التطبيق يستخدم Yahoo Finance API المجاني:
- لا يحتاج API key
- بيانات يومية (Daily candles)
- آخر 6 أشهر من البيانات

## المؤشرات التقنية

### StochRSI
- فترة RSI: 14
- فترة Stochastic: 14
- تنعيم K: 3
- تنعيم D: 3
- إشارة شراء: عبور فوق 20%
- إشارة بيع: عبور تحت 80%

### MACD
- EMA سريع: 12
- EMA بطيء: 26
- خط الإشارة: 9
- إشارة شراء: MACD يعبر فوق Signal
- إشارة بيع: MACD يعبر تحت Signal

## قاعدة البيانات

### جدول signals
- تخزين جميع الإشارات
- حذف تلقائي للإشارات الأقدم من 30 يوم

### جدول monitoring_status
- حالة كل زوج
- آخر فحص
- آخر إشارة

### جدول settings
- إعدادات التطبيق
- تفعيل/تعطيل الإشعارات
- وقت الفحص اليومي

## الأمان والخصوصية

- ✅ لا يوجد تتبع للمستخدم
- ✅ لا يوجد إرسال بيانات لخوادم خارجية
- ✅ جميع البيانات محفوظة محلياً
- ✅ لا يحتاج حساب مستخدم

## الأذونات المطلوبة

- `INTERNET`: لجلب البيانات من Yahoo Finance
- `WAKE_LOCK`: للفحص في الخلفية
- `VIBRATE`: للاهتزاز عند الإشعار
- `POST_NOTIFICATIONS`: لإرسال الإشعارات (Android 13+)
- `FOREGROUND_SERVICE`: لخدمة الخلفية
- `RECEIVE_BOOT_COMPLETED`: للتشغيل عند بدء الجهاز
